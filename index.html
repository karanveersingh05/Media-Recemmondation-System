Movie Recommendation System

Inspired by Netflix's content-based recommendation engine, this project suggests Hollywood movies by analyzing their features. It uses a TF-IDF (Term Frequency-Inverse Document Frequency) vectorizer from scikit-learn to convert textual data like genres, keywords, and cast into numerical vectors. Cosine similarity is then calculated between these vectors to find and recommend movies with the most similar content to what a user likes. The entire system is built as a standalone Jupyter Notebook without external modules for simplicity.

Key Tools: pandas, numpy, scikit-learn (TfidfVectorizer, cosine_similarity), difflib

Music Recommendation System

This project mirrors Spotify's approach to music discovery by using collaborative filtering. It analyzes the listening history of a large user base to find patterns. The core logic is organized into a reusable module for better code management, containing classes for both a popularity-based model (recommending what's trending) and an item-similarity model (recommending songs liked by similar users). This modular design makes the recommendation engine clean and easy to maintain.

Key Tools: pandas, numpy, and a custom Recommenders.py module with : Set Operations, Jaccard Similarity Index, Co-occurrence Matrix, Collaborative Filtering
